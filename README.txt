###########     Installation Steps    ##########

1. Download zip file and Unzip file on your local server.
2. Put this file inside "c:/xampp/htdocs/".
3. Database Configuration
	i)   Open phpmyadmin
	ii)  Create Database named exam
	iii) Import database exam.sql from downloaded folder(inside Online-Quiz folder)
4. Open Your browser paste the link  "http://localhost/Project Folder Name/" (ie, "http://localhost/Online-Quiz/")
